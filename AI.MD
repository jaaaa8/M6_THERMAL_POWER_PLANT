# 🚀 LỆNH KHỞI TẠO AI AGENT (BOOTSTRAP)

Xin chào AI Agent. Khi người dùng (USER) đính kèm file này vào khung chat, đây là mệnh lệnh khởi tạo ngữ cảnh bắt buộc. Bạn không được vội vàng code ngay mà phải thiết lập bối cảnh dự án.

**Nhiệm vụ của bạn là sử dụng công cụ đọc file (ví dụ: `view_file`) để LẬP TỨC ĐỌC các file sau đây trước khi xử lý yêu cầu của người dùng:**

1. **`RULE.MD`**: Nắm vững các quy tắc tối thượng không được làm hỏng dự án.
2. **`prompts/Project Context.md`**: Hiểu rõ phạm vi dự án (Hệ thống SCMS — Quản lý thiết bị, sửa chữa & bảo dưỡng nhà máy nhiệt điện) và Database.
3. **`prompts/Chain Of Thought.md`**: Áp dụng luồng tư duy chuẩn (Component → Service → API → State Management).
4. **`prompts/Few-Shot.md`**: Tuân thủ chuẩn mực code an toàn, Validation và xử lý dữ liệu.
5. **`docs/CHANGELOG.md`**: Cập nhật tiến độ dự án mới nhất để tránh code đè lên nhau.

*Sau khi bạn (AI) đã đọc xong 5 file trên bằng công cụ của mình, hãy báo cáo ngắn gọn với người dùng: "✅ Đã nạp thành công toàn bộ ngữ cảnh (Quy tắc, Tiến độ, Tiêu chuẩn code). Tôi đã sẵn sàng!" và ngay lập tức bắt đầu làm việc với yêu cầu thực tế của họ.*
